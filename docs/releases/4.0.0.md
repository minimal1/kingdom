# Kingdom 4.0.0 Release Notes

## Summary

`4.0.0`은 Kingdom을 Claude 전용 런타임에서 **multi-engine runtime**으로 확장한 릴리스다.

핵심 변화:

- `claude` / `codex` 실행 엔진 선택 지원
- `CLAUDE.md` 자산의 `AGENTS.md` 병행 배치
- package의 `agents/`, `skills/`를 `.claude/`와 `.codex/`로 동기화
- `supported_engines` 기반 장군 호환성 관리
- 주요 builtin 장군의 Codex 자산 추가

## Major Changes

### Runtime

- `config/system.yaml`의 `runtime.engine`으로 기본 엔진 선택
- `bin/lib/runtime/engine.sh` 도입
- `spawn-soldier.sh`, `petition-runner.sh`, `setup.sh`, `check-prerequisites.sh`가 engine abstraction 사용

### Instruction Files

- `config/workspace-claude.md`를 workspace의 `CLAUDE.md`, `AGENTS.md`로 모두 배치
- `general-claude.md`, `general-codex.md` 자산 지원

### Builtin Generals

#### Dual-engine

- `gen-herald`
- `gen-briefing`
- `gen-doctor`
- `gen-pr`
- `gen-catchup`
- `gen-test-writer`

#### Harness

- `gen-harness-querypie-mono`

## Compatibility Notes

- 기존 Claude 워크플로우는 유지된다
- Codex는 best-effort resume token 기반으로 동작한다
- Slack 경로는 계속 Socket Mode 전용이다
