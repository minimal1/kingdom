Claude Code의 역할

Kingdom의 손과 두뇌

# 사람이 하는 코드 리뷰와 동일한 과정을
# Claude Code가 도구를 써서 수행한다

claude -p "PR #423를 리뷰해줘"

  📖 코드를 읽고 이해한다
  ✏️  리뷰 코멘트를 작성한다
  🧪 테스트를 돌린다
  💬 결과를 보고한다

-p 모드: 사람 없이 프롬프트만 받아 자율 실행

Claude Code 활용 3계층

1
실행 엔진 — 병사
파일을 읽고, 코드를 수정하고, 테스트를 돌리는 실제 작업 수행
2
플러그인 시스템 — 장군의 무기
friday(PR 리뷰), sunday(Jira 대응) 등 전문 플러그인으로 능력 확장
3
Soul 시스템 — 업무 매뉴얼
CLAUDE.md로 팀 컨텍스트 주입. 우리 팀을 이해한 상태에서 일을 시작

AI 도구 vs Kingdom

일반 AI 도구 Kingdom
시작 사람이 질문 이벤트가 자동 트리거
실행 답변을 사람이 적용 Claude Code가 직접 수행
맥락 매번 설명 Soul 시스템으로 항상 기억
확장 프롬프트 엔지니어링 장군 패키지 추가
소통 화면 안에서만 Slack으로 팀과 대화
